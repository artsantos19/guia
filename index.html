Frontend Backend Quiz Recursos

Frontend

O Frontend é a parte do desenvolvimento que lida com a interface do usuário e a interação visual.

  • JavaScript: linguagem dominante para elementos interativos.
  • Frameworks: React, Vue.js, Angular, Svelte
  • TypeScript: adição de tipagem estática ao JavaScript.
  • HTML & CSS: estrutura e estilo das páginas web.

Backend

O Backend é responsável pela lógica de negócios, banco de dados e comunicação com o cliente.

  • Node.js: uso de JavaScript no servidor.
  • Frameworks: Express.js, Django, Flask, Spring Boot
  • Python: versátil, popular para backend e ciência de dados.
  • Java: robusto, usado em sistemas empresariais.
